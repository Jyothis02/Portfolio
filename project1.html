Automatic Rain Detector and Cloth Protection System

This project is designed to automatically detect rain using a rain sensor and protect clothes that are being dried outdoors. The core components include an Arduino Uno, a rain sensor module, and a servo motor. When the sensor detects rainfall, it sends a signal to the Arduino, which then activates the servo motor to pull a cover over the clothes, preventing them from getting wet.

The system offers an efficient, hands-free solution to protect laundry in unexpected rain conditions. Itâ€™s particularly useful in areas with sudden weather changes, helping to avoid re-washing or damage to clothes.

Arduino Code

#include <Servo.h>

Servo tap_servo;
int sensor_pin = 4;
int tap_servo_pin = 5;
int val;

void setup() {
  pinMode(sensor_pin, INPUT);
  tap_servo.attach(tap_servo_pin);
}

void loop() {
  val = digitalRead(sensor_pin);
  if (val == 0) {
    tap_servo.write(0);
  }
  if (val == 1) {
    tap_servo.write(180);
  }
}
